# prismify-ui documentation

Vue 3 component library for admin UIs: SCSS in components, theming via CSS variables (`--pf-*`), Nuxt 4 compatible.

**Current version:** `0.1.0-alpha.x` (alpha). A stable API is expected in `1.0.0`.

**Repository:** [koirodev/prismify-ui](https://github.com/koirodev/prismify-ui)

| [Library development](./library-development.md) | Build, tests, Storybook, versioning |
